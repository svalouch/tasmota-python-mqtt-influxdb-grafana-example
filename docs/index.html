tasmota-python-mqtt-influxdb-grafana-exampleÂ¶

This guide provides a starting point for wiring up an IoT-device running Tasmota to an InfluxDB over MQTT and displaying the results in Grafana.

The guide fokuses on measurement data provided by the Tasmota firmware running on a suitable device. That means that running this on a device like the Sonoff S20 will only yield wifi metrics. A Sonoff POW R2 will export power measurements, too, which is of much higher interest in general.
